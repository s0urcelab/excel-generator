# excel-generator
🖨️ a tool to generate EXCEL

## Usage
recommended nodejs version 11+

#### edit `app.js`
```javascript
// OA token
const OA_TOKEN = 'xxxxxxxxxxxxxxx';
// your name
const USER_NAME = '小明';
```
#### put your PDF into `/pdf` folder

#### build EXCEL
```javascript
npm i
npm run build
```

## Develop
```javascript
npm i
npm start
```
